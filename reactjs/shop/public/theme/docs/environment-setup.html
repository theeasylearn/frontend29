Environment Setup

To get started with FreshCart Theme, the only requirement is a Node Js, Gulp, SASS & Bootstrap environment.

Please note: this step is absolutely optional. It aims for advanced user who want to speed up development process with professional grade Front-End toolset shiped with FreshCart. If you are not familiar with these tools and have no time to dive in you can still use plain HTML / CSS / JS to customize FreshCart. Files you need are located inside FreshCart/dist folder. In this case you can skip this and next sections. Happy coding :)

Modern development toolset.

Below are the tools which we use to building a FreshCart theme. For more detail of the toolset, go to their official website.

Node Js

Node.js is an open-source, cross-platform, back-end JavaScript runtime environment that runs on the V8 engine and executes JavaScript code outside a web browser.

Gulp Js

A toolkit to automate & enhance your workflow. Gulp is an open-source JavaScript toolkit used as a streaming build system in front-end web development.

SASS

Sass (short for syntactically awesome style sheets) is a preprocessor scripting language that is interpreted or compiled into Cascading Style Sheets (CSS).

Bootstrap 5

Bootstrap is a free and open-source CSS framework directed at responsive, mobile-first front-end web development. FreshCart is built on 5 and heavily depends on it.
